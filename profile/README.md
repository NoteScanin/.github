## 📖 About Us

**NoteScanin** is an AI-powered platform designed to solve the struggles of digitizing handwritten notes. Say goodbye to unreadable handwriting ("tulisan cakar ayam"), time-consuming manual transcription, and the general struggle of managing student or professional notes!

Our mission is to make note-taking and knowledge sharing as effortless as possible.

## 🌟 Our Core Repositories

We follow a microservices architecture. Here are our main repositories:

- **[Frontend](https://github.com/NoteScanin/frontend)**: The user interface built with Next.js, providing a fast and responsive experience.
- **[Backend](https://github.com/NoteScanin/backend)**: The core API server built with Node.js and Express.
- **[AI Service](https://github.com/NoteScanin/ai)**: The brains of NoteScanin, utilizing TrOCR and IndoBERT for robust Optical Character Recognition (OCR).
